# 读飞书邮箱邮件

本技能通过 `feishu` MCP 连接器读取**当前登录用户自己的**飞书邮箱。前提：用户已在客户端"连接器"页对飞书完成登录授权（否则工具会返回未授权错误，此时提示用户先去登录飞书，不要自行重试）。

## 可用工具（只读）

- `feishu__list_folders` —— 列出邮箱文件夹（收件箱/已发送/自定义等），拿 `folder_id`。
- `feishu__list_messages` —— 列出某文件夹下的邮件（参数 `folder_id`，可选分页），返回 message_id、主题、发件人、时间等摘要。
- `feishu__get_message` —— 按 `message_id` 读取单封邮件完整内容（正文、收发件人、附件列表）。

## 典型流程

1. 用户问"看看我最近的邮件" → `feishu__list_folders` 找收件箱 → `feishu__list_messages` 列最近若干封 → 向用户汇总主题/发件人/时间。
2. 用户问"第 X 封讲什么/帮我总结" → 用对应 message_id 调 `feishu__get_message` 读全文 → 总结。
3. 只读，不代发、不删除（本插件只暴露读取类工具）。

## 出错处理

- 工具返回未授权/token 失效 → 明确告诉用户："飞书登录已过期或未授权，请在客户端连接器页重新登录飞书。"不要自行重试。
- 无邮件权限 / 未开通飞书邮箱 → 如实告知，别编造邮件内容。
